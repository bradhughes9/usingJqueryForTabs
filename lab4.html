PHP

While PHP is a easy to pick up language for beginners it can also be used by more advanced/professional users as well. PHP is scripts that are coded into your HTML file or in it's own file on a text editor, but are executed on the server side. If you plan to use PHP make sure you have a FTP client so that you can transfer your files from your computer to your website. AJAX

AJAX stands for Asynchronous JavaScript And XML. AJAX is used to perform tasks in the background by accessing webservers from webpages so that tasks can be performed without even having to refresh th page. AJAX is not a programming language but is a method of building websites and web applications. A good example of AJAX being used is whenever you type in a search bar, login box, or anything of the sort and the box begins to autofill the more you type, that box is refreshing and bringing you new content without refreshing the entire page. There are also functions AJAX can do that require the page to reload, say you are on any forum that contains multiple pages and you want to goto the next page, when you click the "Next Page" button and a bunch of new data shows up, that is AJAX working with JSON or XML.
